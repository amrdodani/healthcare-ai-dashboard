
# Healthcare GPT Platform

This is a complete AI-powered platform to support healthcare operations. It uses GPT to automate and improve:

- Patient experience analysis
- Discharge planning
- Capacity & resource management
- Strategic planning and KPI tracking
- Task management from AI insights

## Getting Started

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Run the app locally

```bash
streamlit run app/dashboard.py
```

### 3. Or deploy via Docker

```bash
docker-compose up --build
```

## Modules

- `Patient Experience Analyzer`
- `Discharge Planner Assistant`
- `Capacity & Admission Manager`
- `Strategic Planning Assistant`
- `AI Task Tracker`

Each is accessible via tabs in the dashboard.
